# Party AI

A minimalistic, modern Single Page Application for Party AI startup website, built with Vite and React.

## Features

- Clean, user-friendly design
- Embedded Google Form for user interaction
- Fast development with Vite
- Responsive layout

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5174](http://localhost:5174) in your browser.

## Build for Production

```bash
npm run build
```

## Technologies Used

- React 19
- Vite
- ESLint
